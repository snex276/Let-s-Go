# Let's-Go Website

This is the homepage for Let's-Go, a transportation service for foreigners visiting Sri Lanka.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Contact form with basic validation
- Modern, user-friendly interface
- Sections for services, about, and contact information

## Setup

1. Place your logo file as `logo.png` in the root directory.
2. Open `index.html` in a web browser to view the site.
3. For development, use a local server (e.g., VS Code Live Server extension) to avoid CORS issues with external images.

## Customization

- Edit `styles.css` to change colors, fonts, or layout.
- Modify `index.html` to update content.
- Add functionality to `script.js` as needed.

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
